Métodos de instalación

Elige el método que mejor se adapte a tu proyecto:

Con npm

Ideal para proyectos modernos con herramientas como Vite, Webpack o Next.js.

npm install -D tailwindcss
npx tailwindcss init

Con CDN

Para pruebas rápidas sin configuración.

<script src="https://cdn.tailwindcss.com"></script>

Con CLI

Útil para quienes no quieren usar herramientas externas. Sólo necesitas configurar un archivo CSS

npx tailwindcss -i ./input.css -o ./output.css --watch

Con Frameworks & CMS

Tailwind se integra fácilmente con Laravel, Nuxt, WordPress y más

composer require laravel-frontend-presets/tailwindcss npm install && npm run dev

Con Node.js

Recomendado para desarrollos profesionales con herramientas modernas como Vite, Webpack o Next.js.

  1. Instala Tailwind vía npm: 
    npm install -D tailwindcss
npx tailwindcss init
  2. Configura el archivo tailwind.config.js según tus preferencias.
  3. Crea tu archivo CSS: 
    /* styles.css */
@tailwind base;
@tailwind components;
@tailwind utilities;
  4. Compila el CSS con CLI: 
    npx tailwindcss -i ./styles.css -o ./output.css --watch
