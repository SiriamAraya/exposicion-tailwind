Métodos de instalación

Elige el método que mejor se adapte a tu proyecto:

1. Vite Plugin

Usa el plugin de Tailwind con Vite para proyectos modernos con frameworks como React, Vue, Svelte o Laravel.

npm create vite@latest my-project
cd my-project

npm install tailwindcss @tailwindcss/vite

Agrega el plugin en vite.config.ts

import { defineConfig } from 'vite'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [tailwindcss()],
})

Importa Tailwind en tu archivo CSS:

@import "tailwindcss";
Documentación Vite Plugin

2. PostCSS Plugin

Ideal si tu proyecto ya usa PostCSS (como en Next.js o Angular). Tailwind se integra como plugin de PostCSS.

npm install tailwindcss @tailwindcss/postcss postcss

Agrega el plugin en postcss.config.mjs:

export default {
  plugins: {
    "@tailwindcss/postcss": {},
  }
}

Importa Tailwind en tu CSS:

@import "tailwindcss";
Documentación PostCSS Plugin

3. CLI Tool

Instala Tailwind como herramienta de línea de comandos. Ideal para proyectos pequeños o sin framework.

npm install tailwindcss @tailwindcss/cli

Crea un archivo src/input.css con:

@import "tailwindcss";

Genera tu archivo CSS compilado:

npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
Documentación CLI Tool

4. Play CDN

No instalas nada. Solo agregas un <script> a tu HTML y puedes empezar a usar clases de Tailwind directamente

<script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script>

También puedes usar estilos personalizados con <style> type="text/tailwindcss</style>

<style> type="text/tailwindcss</style>
Documentación para usar con Play CDN
