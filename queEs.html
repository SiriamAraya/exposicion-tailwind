¿Que es Tailwind CSS?

Tailwind CSS es un framework de CSS basado en utilidades (utility-first) que permite aplicar estilos directamente en las clases del HTML, en lugar de escribir CSS desde cero, se puede aplicar clases predefinidas que representan estilos específicos, lo que acelera el proceso de desarrollo y mejora la consistencia visual de tus proyectos.

Algunas Características principales de Tailwind

Basado en utilidades

Cada clase aplica un único estilo, brindando control granular, eso significa que cada clase posee propiedades específicas, lo que te da control más preciso:


<button class="bg-blue-500 text-white p-4 rounded-lg">
  Click aquí
</button>

Este es un botón de ejemplo utilizando Tailwind CSS y posee estas Características

  • bg-blue-500>bg-blue-500 → color de fondo azul.
  • text-white → color de texto blanco.
  • p-4 → padding de 1rem (16px) en todos los lados.
  • rounded-lg → bordes redondeados.

Altamente personalizable

Puedes modificar el archivo tailwind.config.js según tus necesidades. Este archivo te permite definir tus propios colores, fuentes, tamaños, etc. para que todo el equipo use los mismos valores.

tailwind.config.js

module.exports = {
  theme: {
    extend: {
      colors: {
        brand: '#ff6600', // color corporativo
      },
      spacing: {
        '72': '18rem',   // nuevo tamaño de margen/padding
      }
    }
  }
}

Html


<div class="bg-brand p-72">
  Caja con color corporativo y padding enorme
</div>

Estas propiedad puestas en el archivo tailwind.config.js se reflejan en el HTML de esta manera:

  • bg-brand aplica background-color: #ff6600.
  • p-72 aplica padding: 18rem en todos los lados.

CSS optimizado

A diferencia de Bootstrap, Tailwind crea un CSS más ligero y eficiente al eliminar estilos no utilizados durante el proceso de construcción. Esto se configura en el archivo tailwind.config.js

Ejemplo

Supongamos que tienes el siguiente HTML:


<button class="bg-red-500 p-4">Botón</button>

Entonces el CSS final solo contendrá las clases bg-red-500 y p-4, ahorrando muchísimo peso.
