Cómo configurar paleta de colores, tamaños y tipografía en un proyecto con Tailwind CSS

1. Paleta de colores

En Tailwind puedes definir tus propios colores usando @theme para reutilizarlos en toda la página

@import "tailwindcss";
@theme {
  --color-primario: #6200ee;
  --color-primario-dark: rgb(98, 0, 238);
  --color-secundario: rgba(255, 0, 150, 0.8);
}

Ejemplo: Texto con color primario

2. Tamaños personalizados

También se pueden definir medidas para usar en márgenes, paddings, anchos, altos, etc

@import "tailwindcss";
@theme {
  --text-tamanno1: 1.2rem;
  --text-tamanno2: 1.3rem;
  --text-tamanno3: 1.5rem;
}

Ejemplo: Texto con tamaño de letra 3

3. Tipo de letra

Agrega fuentes para títulos, textos y código

@import "tailwindcss";
@theme {
  --font-tipoLetra1: "Inter", sans-serif;
  --font-tipoLetra2: "Merriweather", serif;
  --font-tipoLetra3: "Courier New", monospace;
}

Ejemplo: Texto con fuente Inter
