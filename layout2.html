Layout en Tailwind CSS - Ejemplos Interactivos

1. Containers

El contenedor container centra el contenido y ajusta su ancho según el tamaño de la pantalla para mantener una lectura cómoda. Es útil para limitar el ancho máximo y evitar que el contenido se expanda demasiado en pantallas grandes.

Este es un contenedor que cambia su ancho según la clase aplicada.

Prueba cambiar su tamaño con los botones arriba

2. Grid

El sistema de grid permite dividir el espacio en filas y columnas fácilmente. Puedes controlar la cantidad de columnas con clases como grid-cols-2, grid-cols-3, etc. El espaciado entre celdas se controla con gap-*.

3. Flexbox

Flexbox es un sistema de diseño flexible que permite alinear y distribuir elementos dentro de un contenedor. Cambia la dirección con flex-row o flex-col y la alineación con justify-* y items-*.

4. Spacing

El espaciado (spacing) es fundamental para la separación visual. Tailwind usa clases como p-4 (padding) y m-4 (margin). También puedes controlar la separación entre elementos con gap-*.
