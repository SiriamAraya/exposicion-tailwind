Tema Oscuro

Elige entre el modo de control manual con botón o el modo automático según tu sistema.

Variante `dark`

Tailwind incluye una variante llamada `dark` que permite diseñar el estilo de la página cuando el modo oscuro este activo

Ejemplo

<div class="bg-white dark:bg-gray-800 rounded-lg px-6 py-8 ring shadow-xl ring-gray-900/5">
  <p class="bg-gray-300 text-black dark:bg-gray-600 dark:text-white p-2 text-lg text-center">Ejemplo</p>
</div>

Activar y desactivar el modo oscuro manualmente

Las utilidades se aplicacan cada vez que la clase dark se añade al elemento <html>. Puedes alternar el modo oscuro manualmente con un botón

Uso de un atributo de datos

También puedes controlar el modo oscuro usando un atributo en lugar de una clase. Por ejemplo, estableciendo data-theme="dark" en la etiqueta <html>, las utilidades dark

Con soporte para temas del sistema

También puedes que se adapte automáticamente a la configuración del tema del sistema operativo. Si tu sistema está en modo oscuro, verás los estilos oscuros; si está en modo claro, verás los claros. Para ver la diferencia, cambia la configuración de tu sistema operativo.
